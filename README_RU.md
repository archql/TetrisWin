# Тетрис Вин v.6.6.* Мемный апдейт/3D версия. Релиз.
## Краткое описание
С этой версией тетриса Вы теперь можете не просто расширить горизонты, Вы получаете 3е измерение для игры*!
Вместе с этим с возмижностью изменять игровые текстуры и звуки Ваше воображение может играть без границ!
И все это во время игры с друзьями в соревновательном режиме!

### Краткий список отличий версии 6.6.\* и 5.10.\*
- управление мышью
- заменяемые музыкальные эффекты и текстуры (прямо во время игры!)
- работа в режиме прокси
- 3D вращение поля
- поддержка Виндовс 11
- исправления
- размер вырос с `12 кб` до `16 кб`.

### 📶СЕТЕВОЙ РЕЖИМ 📶

#### У клиента есть 5 состояний:
 - `OFFLINE` &mdash; клиент не подключён к сети.
 - `ON-LINE` &mdash; клиент подключился к сети, но регистрация еще не завершилась.
 - `REGSTRD` &mdash; клиент зарегистрировался в сети и теперь может полноценно взаимодействовать с другими клиентами 
 - `REJCTED` &mdash; клиент пытался подключиться к сети, в которой уже есть клиент с тем же именем.
 - `UUIDERR` &mdash; клиент пытался получить UUID устройства, но не смог. Состояние идентично состоянию `OFFLINE`.
 
#### Алгоритм подключения:
 - **Используй клавишу *F3* чтобы подключиться к локальной сети** (для этого ваше текущее состояние `OFFLINE` или `REJCTED`). Если новое состояние `ON-LINE` &mdash; все прошло хорошо. 
 - Далее клиент отправляет запрос на регистрацию. Если следующее состояние `REGSTRD` &mdash; все прошло хорошо. 
 - Если вы получили состояние `REJCTED` &mdash; Смени никнейм и попытайся снова (Используй клавишу **F3**).
 - Подключение к сети так же используется для обновления таблицы лидеров.
 - Клиент все еще получает обновления таблицы лидеров в состоянии `REJCTED`.
 - Просмотр клиентов онлайн (**ПКО**). Каждый клиент в сети отправляет кадр игры с сообщением пинг. 
 - Остальные клиенты отображают их.
 
#### Соревновательный режим игры:
 - **Соревновательный режим игры** устанавливает клиентам одинаковый ключ случайного генератора и запускает игру
 - используй клавишу **F4** для запуска **у каждого клиента в сети!**. Игра начнётся только в состоянии `GAME OVER` и `REGSTRD`
 
#### Другие особенности игры по сети:
 - ⚠️**ВНИМАНИЕ. Используемый сетевой протокол не гарантирует доставку сообщений!**
 
### 🎨ГРАФИКА🎨

 - по умолчанию, клиент рисует в 3D используя цветные кубики 
 - на кубики может быть натянута текстура, для этого поместите *\*.bmp* файлы с *любым* именем в папку с игрой
 - ⚠️**ВНИМАНИЕ! BMP должен быть в формате *R8* *G8* *B8* (*24 бит*) и размером менее 100 кб!**
**Использование неправильного формата вызовет падение приложения**
 - клиент использует BMP автоматически, когда они находятся в папке с игрой
 - можно поместить несколько BMP в папку с игрой и переключать текстуры клавишей **F5**
 - игровое поле может вращаться по оси Z если удерживать shift (или оси X если удерживать ctrl).
 
### 🎵 МУЗЫКА 🎵
 - по умолчанию, вся музыка реализована через миди
 - звуки можно поменять, для этого поместите *\*.wav* файлы с *специальным* именем в папку с игрой
 - `специальное` имя должно быть в формате: код заменяемого миди-сообщения в шестнадцеричной системе с символом '#' впереди и расширением файла '.wav' 
 
### 💠Заменяемые звуки и текстуры (v6.6+)💠
 - Все заменяемые звуки должны быть в формате .wav
 - звук конца игры будет случайным и выбран из 3-х представленных (правила ниже)
 - дополнительно звук может менять текстуру, имя которой отформатированно по правилам: код заменяемого миди-сообщения в шестнадцеричной системе с символом '#'
 и расширением файла .bmp)
 
| **миди-сообщения с символом '#'**    | **Когда проигрываются?**|
| ------- | ------------------ | 
| #7F2C93 | 1 линия уничтожена |
| #7F3093 | 2 линии уничтожены |
| #7F3493 | 3 линии уничтожены |
| #7F3893 | 4 линии уничтожены |
| #7F2594 | звук взрыва при уничтожении 4-х линий (конфликтует с предыдущим) |
| #7F2690 | звук конца игры 1 |
| #7F2290 | звук конца игры 2 |
| #7F1E90 | звук конца игры 3 |

### 📑Сохранение прогресса игры📑

- весь игровой процесс сохраняется в файле `.ttr`. 
- ⚠️**ВНИМАНИЕ**⚠️ *Полный файл`.ttr`* является единственным подтверждением вашего счёта! Вы обязаны не изменять и не передавать его 3-м лицам для защиты вашего результата! Для дополнительной защиты храните этот файл отдельно от игрового клиента.
